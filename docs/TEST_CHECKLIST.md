# Test Checklist

## Happy path
1. Новый пользователь -> `🎁 Пробный период` -> выбор сервера -> триал создан на 3 дня.
2. В конце триала пользователь получает уведомление о покупке.
3. Пользователь создает счет `month` через YooKassa -> webhook `succeeded` -> подписка активируется.
4. Пользователь создает счет `year` через Crypto mock -> webhook `paid` -> подписка активируется.
5. `📦 Моя подписка` показывает корректный срок.
6. За 24 часа до конца подписки приходит уведомление.
7. После конца подписки аккаунт в s-ui удаляется.

## Edge cases
1. Повторный запрос триала -> отказ.
2. Невалидный ID сервера -> ошибка без падения.
3. Повторный webhook payment -> идемпотентный результат (дубль не создает вторую подписку).
4. Неуспешные статусы webhook -> игнор.
5. Пользователь не оплатил после триала -> удаление через +1 день.
6. Админ-команды недоступны не-админам.
7. Антифлуд ограничивает спам сообщений.
8. При недоступности s-ui задача повторно запускается без порчи данных.
